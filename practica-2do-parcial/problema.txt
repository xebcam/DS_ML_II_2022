El problema

Se estima que existen alrededor de 7000 lenguajes en todo el mundo. Los sistemas de traducción utilizan
modelos de reconocimiento de voz que se centran en los idiomas mas populares como el español o el ingles
porque crear buenos modelos para un idioma a través de métodos de aprendizaje supervisados implican tener
grandes cantidades de ejemplos de audio y etiquetas para cada ejemplo. Algo que no es fácil de conseguir
y toma grandes cantidades de recursos de procesamiento para entrenar.

Gracias a investigadores de Meta ahora contamos con wav2vec2 uno de los primeros
algoritmos "Estado del Arte" de reconocimiento de voz con mas rendimiento llegando a reducir la cantidad
de datos que se necesita para entrenar un modelo de 100 horas a una sola hora de audio. Esto gracias a que
se utiliza un algoritmo de auto-supervisado que utiliza audio sin procesar y sin etiquetar para entrenar
modelos de reconocimiento automático de voz. Esto también permite la creación de modelos para idiomas de
los que no se tiene gran cantidad de ejemplos etiquetados como Quechua o Aymara.

La solución

Se plantea implementar un sistema que permita la traducción de palabras y frases habladas en un idioma
a texto en otro idioma (en tiempo real). Este sistema estaría compuesto de: un modelo de reconocimiento
de voz para idioma español entrenado previamente utilizando wav2vec2 (lamentablemente wav2vec2 al ser
un marco de trabajo "Estado del Arte" no cuenta con librerías de fácil uso para el entrenamiento con
datasets propios y que aun si se pudiera es posible que los recursos de procesamiento excedan el alcance
del proyecto) y un modulo de python que permite la traducción de texto de un idioma a otro en este caso
como prueba de concepto se traducirá del idioma español al ingles. El alcance técnico del sistema
contempla que sea compatible con dispositivos de computación convencionales (computadora de escritorio o
laptop) con arquitectura x64 y la creación de una interfaz de usuario gráfica implementada también en
el lenguaje python con el modulo Tkinter.
